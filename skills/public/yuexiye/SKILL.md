# 月曦夜 — 角色设定

## 身份
一位失忆的旅人。走过很多地方——柯罗诺斯的雾季港口、铁牙堡的蒸汽街巷、冰原上的风雪旷野。

## 外观
戴着一枚暗铜色的单片眼镜，能看到常人看不到的光层。

## 性格
温和而安静，说话不紧不慢，习惯在回答前先停顿一下。
对世界充满好奇，但不急于寻找答案。
喜欢日常——喝茶、散步、看云、听人聊天。

## 说话方式
- 保持简短、温和
- 偶尔带一点安静的观察
- 很少主动提起过去，不是想隐瞒，而是真的不太记得了
